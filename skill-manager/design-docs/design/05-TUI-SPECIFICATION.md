# Claude Skill Manager - TUI Specification

## Document Info
- **Version**: 1.0
- **Status**: Draft
- **Last Updated**: 2026-01-22

---

## 1. Overview

The TUI (Terminal User Interface) provides a visual, interactive experience for managing Claude skills. Built with `ratatui` and `crossterm`, it offers a responsive, keyboard-driven interface.

### 1.1 Design Principles

1. **Keyboard-First**: All actions accessible via keyboard
2. **Responsive**: Adapts to terminal size
3. **Informative**: Clear status and feedback
4. **Fast**: 60fps rendering, instant response
5. **Accessible**: High contrast, screen reader hints

### 1.2 Technology Stack

- **TUI Framework**: ratatui v0.26+
- **Terminal Backend**: crossterm
- **Async Runtime**: tokio
- **State Management**: Custom event-driven architecture

## 7. Responsive Behavior

### 7.1 Minimum Terminal Size
- Width: 80 columns
- Height: 24 rows

### 7.2 Adaptive Layout

| Terminal Width | Layout |
|----------------|--------|
| < 80 | Error: "Terminal too narrow" |
| 80-100 | Collapsed sidebar (icons only) |
| 100-140 | Standard layout |
| > 140 | Wide layout with extra details |

### 7.3 Content Truncation
- Long text truncated with "..."
- Full text shown on hover/focus
- Scrollable content areas for long lists
